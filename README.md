# Real-world Data Project (Retail Sales Analysis)

## Objective
Analyze a retail sales dataset and build a machine learning model to predict sales based on available features.

## Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization
- Machine learning model (Linear Regression)

## Visualizations
- Sales by Category
- Monthly Sales Trend
- Correlation Heatmap

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Model Performance
The model achieved high accuracy (~95%) due to strong patterns and structured trends in the dataset (increasing sales and consistent category behavior).

In real-world scenarios, data is more complex and noisy, which would result in lower accuracy.

## Insights
- Technology category has the highest sales
- Sales show an increasing trend over time
- Different categories contribute differently to revenue
- Seasonal patterns can be observed in monthly sales

## Conclusion
This project demonstrates how data science techniques can be applied to real-world datasets to extract insights and build predictive models.
